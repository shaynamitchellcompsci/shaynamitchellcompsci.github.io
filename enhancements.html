Setup Instructions

  • System: This app needs to run in the following environment: Android Studio.
  • Programming Language: Java
  • Database: SQLite – Android feature
  • Dependencies: AndroidX Libraries, SQLite Database
  • Android Studio: Download and install
  • Java Development Kit (JDK): Version 8 or later (Included in Android Studio)
  • Android Emulator or Physical Device: Required for running the app
  • Project Setup Steps:
    • Download the project ZIP file and extract it.
    • MacOS: Double-click the project file to open in Android Studio.
    • Windows: Open Android Studio → Select "Open an Existing Project" → Choose the "Weight" project folder.
    • Wait for Android Studio to sync dependencies.
    • Clean, rebuild, and sync Gradle files.
